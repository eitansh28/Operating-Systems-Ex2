# Operating-Systems-exc2

The purpose of the exercise was to produce a chain of signals. Running the program will result in the creation of 8 signals one after the other. 
Types of signals:
SIGCHLD,
SIGUSR1, 
SIGFPE, 
SIGALRM, 
SIGILL, 
SIGTSTP, 
SIGSEGV,
SIGIO, 
SIGABRT.

Command to compile the program:


>gcc -o main exc2.c


and then:

>./main
